# 👟 Luxe Shoes Website

A modern and responsive landing page for a premium shoe brand, built with **HTML5**, **CSS3**, and **Bootstrap 5**. The website features a clean user interface, modern styling, and a fully responsive layout that works across desktop, tablet, and mobile devices.

## 🚀 Features

* Responsive navigation bar
* Modern hero section
* Featured shoe collection
* Why Choose Us section
* Customer reviews
* Newsletter subscription
* Stylish footer
* Mobile-friendly design
* Smooth hover effects
* Clean and modern UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5

## 📱 Responsive Design

The website is fully responsive and optimized for desktop, tablet, and mobile screens using Bootstrap's grid system.

## 📂 Project Purpose

This project was created to practice building a complete modern landing page while improving Bootstrap, responsive design, and frontend development skills.
